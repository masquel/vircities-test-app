Task
Сверстать страничку с применением адаптивной верстки, которая загружает сообщения из localStorage по тегу из поля ввода.
 
На странице должно находится: 1 поле ввода, 1 кнопка
 
При вводе сообщения в формате "#тег сообщение", сообщение должно сохранятся в localStorage, cообщения должны записываются при нажатии на кнопку или на enter
 
При вводе только одного тега "#тег", связанные с этим тегов сообщения должны динамически считаться из localStorage и вывестись на экран (без нажатия на кнопку или enter)
